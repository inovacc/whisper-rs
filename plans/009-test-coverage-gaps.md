### Step 3: cfg-off download fallback
Create `tests/pipeline_download_off.rs`:
```rust
#![cfg(not(feature = "download"))]
use whisper_rs::pipeline::{ModelRef, Pipeline};

#[test]
fn download_without_feature_is_config_error() {
    let err = Pipeline::builder().whisper_model(ModelRef::download("tiny.en")).build();
    assert!(matches!(err, Err(whisper_rs::WhisperError::Config(_))));
}
```
**Verify**: `cargo test --no-default-features --test pipeline_download_off` → passes; and confirm it is inert
(compiles to nothing) with the feature on: `cargo test --features download` still green.
